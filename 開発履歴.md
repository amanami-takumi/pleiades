- yfinance/Yahoo Finance 側のレート制限対策として、リクエスト間隔、銘柄間隔、レート制限時のバックオフ再試行を追加。
- 投資情報チャートに縦軸の価格ラベルと横軸の日付ラベルを追加。
- 非上場投資信託向けに Yahoo!ファイナンス時系列 HTML のフォールバック取得を追加。
- 投信協会コード形式の `asset_type=fund` は yfinance を先に叩かず、Yahoo!ファイナンス HTML から基準価額を保存するよう変更。

- BFF API が利用できない場合は従来の HTML 表スクレイピングへフォールバックする構成に変更。

- 購入履歴テーブルと API/UI を追加し、複数回購入の投資額、保有数量、評価額、評価損益を表示。
